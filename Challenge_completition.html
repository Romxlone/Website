      inicio con CSS inline - internal - external
        °inline va en las tags con la property ej -> <body style="background: red"> , se recomienda usar solo en secciones especificas
        °internal va dentro del tag style con el elemento ej -> <style> html{ backround: red; } </style>, se recomienda para paginas unicas (no multipaginas)
        °External es cuando usamos otro archivo styles.css ej -> <link rel="stylesheet" href="./styles.css" />, es el mas recomendcado para sitios web multipaginas
      css selectors - es la parte que selecciona el elemento de html para modificar ej -> h1{ color : blue}
        Class selector modifica el estilo de la clase (que es un atributo de una etiqueta) del elemento(etiqueta) html ej-> .nombre-clase { color:red;}, uso: se aplica a varios elementos
        Id Selector modifica el estilo id(que es un atributo de una etiqueta) de una etiqueta ej -> #id-name {color: red;}, uso: se aplica a un elemento dentro de un archivo html
        Attribute Selectormodifica el estilo del attributo de la etiqueta html ej -> p<-(etiqueta) p[draggable="true"]{color: red}
        Universal Selector Selecciona todo donde el stylesheet este activo ej-> *{ color: red;}
