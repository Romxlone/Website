      inicio con CSS inline - internal - external
        inline va en las tags con la property ej -> <body style="background: red"> , se recomienda usar solo en secciones especificas
        internal va dentro del tag style con el elemento ej -> <style> html{ backround: red; } </style>, se recomienda para paginas unicas (no multipaginas)
        External es cuando usamos otro archivo styles.css ej -> <link rel="stylesheet" href="./styles.css" />, es el mas recomendcado para sitios web multipaginas
